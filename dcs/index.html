Chronos

A lightweight, zero-dependency terminal UI for exploring Git repository analytics.

Chronos is a Ruby TUI (Text User Interface) that gives you a live, interactive dashboard of your Git project's commit activity. It runs entirely in the terminal with no external dependencies, no bundler, and no configuration.

Why Chronos?

Most Git analytics tools require a web server, a database, or a SaaS subscription. Chronos cuts through all of that — it's a single process that reads your Git history and displays it immediately in a clean, navigable terminal interface.

Quick Start

git clone https://github.com/BobaSipp/chronos.git
cd chronos
ruby chronos.rb

Run it from any Git repository directory. Switch between tabs with ← and →, scroll with ↑ and ↓, and quit with q.

How It Works

Chronos runs git log --name-only --oneline to parse commit history, groups changed files, and ranks them by frequency. Author stats come from git log --format=%an. All rendering is done with raw ANSI escape codes — no curses or terminfo required. The entire application is a single logical unit organized into clean Ruby modules under lib/chronos/.
