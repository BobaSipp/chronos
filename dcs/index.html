Chronos

Lightweight terminal-based Git repository analytics dashboard.

Chronos is a Ruby TUI that gives you a live, interactive overview of your Git repository's commit activity. No external dependencies, no bundler, no fuss.

Quick Start

gem install chronos
cd my-git-repo
chronos

Run it from any Git repository. Press Tab to cycle tabs, j/k to scroll, q to quit.

Features

  • Hotspots tab — most-changed files ranked by commits
  • Authors tab — top contributors ranked by commits
  • Info tab — repo overview and file-type breakdown
  • Scrollable lists — j/k navigate long lists
  • Auto-refresh (--watch) — updates every 2s
  • JSON export (--json) — pipe data to other tools
  • Zero dependencies — Ruby stdlib only

How It Works

Chronos runs git log --name-only --oneline to parse commit history, groups changed files, and ranks them by frequency. All rendering is done with raw ANSI escape codes. The application is organized under lib/chronos/.
