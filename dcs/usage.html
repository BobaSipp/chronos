Usage Guide

ruby chronos.rb

Run from any directory inside a Git repository.

Keyboard Controls

KeyAction
← →Switch tabs
↑ ↓Scroll through lists
qQuit

Tabs

TabDescription
HotspotsMost frequently changed files with bar charts
InfoRepo overview: path, commits, branches, file types
AuthorsTop contributors ranked by commit count

Command-Line Options

OptionDescription
--hotspotsStart on Hotspots tab
--infoStart on Info tab
--authorsStart on Authors tab
--watchAuto-refresh every 2 seconds
--sort countSort hotspots by commit count (default)
--sort nameSort hotspots alphabetically
--jsonExport data as JSON and exit
--help, -hShow help

Examples

ruby chronos.rb --authors
ruby chronos.rb --watch
ruby chronos.rb --sort name
ruby chronos.rb --json | jq '.hotspots[:3]'
