Usage Guide

Everything you need to know to use Chronos effectively.

Basic Usage

ruby chronos.rb

Run from any directory inside a Git repository. Chronos will immediately display the TUI dashboard with your repository's analytics data.

Tabs

Chronos has three tabs, switchable with the ← and → arrow keys:

Tab Description
Hotspots Most frequently changed files, ranked by commit count with visual bar charts. Sortable by count or name.
Info Repository overview: path, total commits, branches, author count, and file-type breakdown.
Authors Top contributors ranked by commit count with bar charts.

Keyboard Controls

← → Switch between tabs ↑ ↓ Scroll through long lists q Quit Chronos

Command-Line Options

Option Description
--hotspots Launch directly into the Hotspots tab
--info Launch directly into the Info tab
--authors Launch directly into the Authors tab
--watch Auto-refresh data every 2 seconds
--sort count Sort hotspots by commit count (default)
--sort name Sort hotspots alphabetically by filename
--json Export all data as JSON and exit
--help, -h Show usage information

Examples

Start on the Authors tab

ruby chronos.rb --authors

Auto-refresh mode

ruby chronos.rb --watch

Data refreshes automatically every 2 seconds. Useful for monitoring active repositories or during long-running processes.

Sort hotspots alphabetically

ruby chronos.rb --sort name

Export data as JSON

ruby chronos.rb --json

Output can be piped into other tools:

ruby chronos.rb --json | jq '.hotspots[:3]'

Project Structure

After cloning, the repository is organized as follows: 

chronos/
  bin/chronos           # Executable entry point
  lib/
    chronos.rb          # Main library entry point
    chronos/
      version.rb        # VERSION constant
      ansi.rb           # ANSI escape codes
      data.rb           # Data collection & GitData struct
      cli.rb            # CLI option parsing & JSON export
      tui.rb            # TUI main loop & keyboard input
      renderer.rb       # All rendering & display logic
  dcs/                  # Documentation website
