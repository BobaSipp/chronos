Features

Hotspots Tab

Shows your repository's most frequently changed files, ranked by the number of commits they appear in. Each file shows its commit count. Sort by count (default) or alphabetically with --sort name.

Authors Tab

Lists every committer ranked by commit count.

Info Tab

Repository overview showing directory path, total commits, branch list, unique author count, and a file-type breakdown grouped by extension.

Scrollable Lists

When lists are longer than the screen, use j and k to scroll. A status line shows current position.

Auto-Refresh (--watch)

Re-collects Git data every 2 seconds. Useful for monitoring active repositories.

JSON Export (--json)

Outputs all data as formatted JSON, then exits. Designed for scripting and integration. 

ruby chronos.rb --json

{
  "version": "0.1b",
  "repo_root": "/home/user/project",
  "total_commits": 142,
  "branches": ["main", "dev"],
  "hotspots": [
    { "file": "src/main.rb", "commits": 34 }
  ],
  "authors": [
    { "author": "Alice", "commits": 80 }
  ],
  "file_types": [
    { "extension": ".rb", "files": 45 }
  ]
}

Zero Dependencies

Uses only io/console, English, and json from Ruby's standard library. No gems, no bundler, no npm.
