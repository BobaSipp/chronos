Features

A detailed breakdown of everything Chronos can do.

Hotspots Tab

The Hotspots tab shows your repository's most frequently changed files, ranked by the number of commits they appear in. Each file is displayed with a visual bar chart that scales relative to the top hotspot, plus the raw commit count.

Use the --sort count and --sort name flags to switch between sorting by commit frequency and alphabetical order. 

ruby chronos.rb --sort name

Authors Tab

See who contributes most to your repository. The Authors tab lists every committer ranked by commit count, with proportional bar charts. Useful for understanding team contribution patterns and identifying "bus factor" risks.

Info Tab

The Info tab provides a high-level overview of your repository:

  • Directory — absolute path to the repository root
  • Commits — total commit count across all branches
  • Branches — list of all local branches
  • Authors — number of unique committers
  • Files changed — number of unique files touched in commit history

It also includes a File Types section that breaks down your project by file extension, showing which types of files dominate your codebase.

Scrollable Lists

When the Hotspots or Authors tabs contain more items than fit on screen, use ↑ and ↓ to scroll through the list. A scroll indicator at the bottom shows the current position (e.g., "32 total, showing 1-9 (28%)").

Auto-Refresh (--watch)

In watch mode, Chronos re-collects Git data every 2 seconds automatically. The terminal shows "Refreshing repository data..." during each cycle. Useful for monitoring active development in real time. 

ruby chronos.rb --watch

JSON Export (--json)

The --json flag outputs all collected data as formatted JSON, then exits. This is designed for scripting and integration with other tools: 

ruby chronos.rb --json

{
  "version": "0.3.0",
  "repo_root": "/home/user/project",
  "total_commits": 142,
  "branches": ["main", "dev"],
  "hotspots": [
    { "file": "src/main.rb", "commits": 34 },
    ...
  ],
  "authors": [
    { "author": "Alice", "commits": 80 },
    ...
  ],
  "file_types": [
    { "extension": ".rb", "files": 45 },
    ...
  ]
}

CLI Flags

Chronos supports launching directly into any tab via command-line flags: 

ruby chronos.rb --hotspots   # Start on Hotspots tab
ruby chronos.rb --info       # Start on Info tab
ruby chronos.rb --authors    # Start on Authors tab

Zero Dependencies

Chronos uses only Ruby's standard library modules:

  • io/console — terminal size detection and raw keyboard input
  • English — $CHILD_STATUS alias for $?
  • json — JSON generation for the --json flag

No Gemfile, no bundle install, no npm install, no compilation step. Clone and run.

Portable

Since Chronos is a single logical application with no external dependencies, it runs on any platform that supports Ruby and Git: Linux, macOS, Windows (with Ruby installed), and even inside containers or CI environments.
