Installation

Get Chronos running on your machine in under a minute.

Prerequisites

  • Ruby (MRI Ruby 2.5 or later) — ruby-lang.org
  • Git — any modern version
No external gems required. Chronos uses only Ruby's standard library. You do not need Bundler, RVM, rbenv, or any other tooling.

Setup

  1. Clone the repository 
    git clone https://github.com/BobaSipp/chronos.git
cd chronos
  2. Verify Ruby is installed 
    ruby --version

    You should see output similar to ruby 3.4.0.

  3. Run it 
    ruby chronos.rb --help

    You should see the usage information printed to your terminal.

Tip: Run ruby bin/chronos as an alternative entry point.

Verification

Navigate to any Git repository and run ruby /path/to/chronos/chronos.rb (or add an alias). You should see the Chronos TUI dashboard with your repository's analytics.

Updating

Since Chronos is installed via Git, updating is straightforward: 

cd chronos
git pull

Having trouble? Check the contributing guide or open an issue on GitHub.
