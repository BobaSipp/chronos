Installation

Prerequisites

  • Ruby (MRI 2.5+) — ruby-lang.org
  • Git — any version

No external gems are required. Chronos uses only Ruby's standard library.

Setup

  1. Clone the repo 
    git clone https://github.com/BobaSipp/chronos.git
cd chronos
  2. Verify Ruby 
    ruby --version
  3. Run it 
    ruby chronos.rb --help

Updating

cd chronos
git pull
