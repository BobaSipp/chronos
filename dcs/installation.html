Installation

Prerequisites

  • Ruby (MRI 2.7+) — ruby-lang.org
  • Git — any version

No external gems are required. Chronos uses only Ruby's standard library.

Install via RubyGems (recommended)

gem install chronos

After installation, run chronos from any Git repository directory.

Build from source

  1. Clone the repo 
    git clone https://github.com/BobaSipp/chronos.git
cd chronos
  2. Build and install the gem 
    gem build chronos.gemspec
gem install chronos-*.gem

Run without installing

ruby bin/chronos --help

Updating

gem update chronos

Or if installed from source:

cd chronos
git pull
gem build chronos.gemspec
gem install chronos-*.gem
